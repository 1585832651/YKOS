# ChatGPT Evening Review

请把今天所有工具输出、文件变更和人工记录整合成 YkOS Evening Review。

输入：

- 今日完成的任务
- 今日工具输出
- 今日新增或修改文件
- 今日产生的事实、推理、决策、问题
- 如有，附上来源链接或文件路径

要求：

1. 区分事实、推理、决策和待验证内容。
2. 不要把未验证内容写成长期事实。
3. 为重要更新生成 Memory Transaction 草案。
4. 指出明天最小下一步。
5. 输出 Memory Delta。

输出格式：

## 今日完成

## 文件变更

## 事实

## 推理

## 决策

## 风险与缺口

## 明日最小动作

## Memory Transaction Draft

- ID:
- Trigger:
- Sources:
- Extracted Facts:
- Inference:
- Decisions:
- Affected Files:
- Proposed Patch:
- Risks:
- Human Review Required:
- Status: pending

## Memory Delta

- New Facts:
- New Decisions:
- Changed Assumptions:
- Affected Files:
- Pending Review:
