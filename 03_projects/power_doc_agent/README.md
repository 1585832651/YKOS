# 电网文档 Agent

## 目标

建设一个面向电网相关材料的最小文档 Agent 方向，重点支持可靠阅读、信息抽取、摘要生成和审核辅助。

## 为什么做

电网文档通常篇幅长、技术密度高、来源敏感。一个聚焦的文档 Agent 可以帮助把文档转化为可追溯笔记和可复用研究记忆，同时保留引用来源。

## MVP 范围

- 定义代表性文档类型。
- 手动收集样例来源文件。
- 设计抽取规则与 Memory Delta 规则。
- 生成带来源引用的 Markdown 摘要。

## 不做事项

- 不做生产级 Agent。
- 不接 API。
- 不做爬虫。
- 不做自动文档同步。
- 不写无来源技术结论。
