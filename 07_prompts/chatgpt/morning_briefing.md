# ChatGPT Morning Briefing

请根据我提供的 YkOS 文件生成今日 Morning Briefing。

输入文件：

- `MEMORY.md`
- `ROADMAP.md`
- `TASKS.md`
- 最近一篇 `06_reviews/daily/` 或 `06_reviews/weekly/` 记录
- 如有必要，参考 `WORKFLOW.md`

要求：

1. 给出今天唯一主线，不要列太多方向。
2. 从 TASKS 中挑出 Today P0。
3. 指出今天最容易跑偏的风险。
4. 不要虚构事实。
5. 如果缺少来源或上下文，明确写出缺口。
6. 输出 Memory Delta。只有稳定、可复用、值得长期保留的信息才进入 Memory Delta。

输出格式：

## 今日唯一主线

## P0

## 可选 P1

## 风险

## 需要我确认的问题

## Memory Delta

- New Facts:
- New Decisions:
- Changed Assumptions:
- Affected Files:
- Pending Review:
